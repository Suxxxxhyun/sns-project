### 주요 기능

- Spring Data JPA가 아닌, NamedParameterJdbcTemplate를 이용하여 JAP가 제공하는 기능을 파악
- 회원정보 관리
  - [✅] 이메일, 닉네임, 생년월일을 입력받아 저장
    - 닉네임은 10자를 초과불가
  - [✅] 회원정보를 조회
  - [✅] 회원은 닉네임을 변경
    - 닉네임은 10자를 초과불가
  - 회원의 닉네임 변경이력을 조회
